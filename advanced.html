Header Tag and Use

The main use of the header element is to enclose h1-h6 heading tag inside. It is a semantic or structural tag
Unlike the h1-h6 tags, It does not make text bold or big.
Technically speaking, It is used to indicate that this is the head of a document

Input Tag and it's types

The input tag has 6 main different types of data that can be entered in it's field. Web browsers and Integrated Dev Environments(IDEs) are modified to understandand react accordingly to the type of an input

Navigation Tag and Use

A nav tag is used to build a navigation bar that has a lot of links inside it in an orderly structure. The most common tags you can find in a navigation tag is the DIV and anchor/linking tag
  • Div is used to hold all the text in a enclosed format and make things ordered
  • A or anchor tag is used to link to the pages contained in your website

Ways of Adding Colors with Css

There are three main ways of adding colors using CSS to an html element

  • Using the normal Everyday Color style: Red, Green, Blue, Brown, Black and so on.
    • Example p{ color: red; }
  • Using The Hex-Values of Adding Color. This method offers more color ranges and combinations Example p{ color: #696969; }
