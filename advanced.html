Header Tag and Use

The main use of the header element is to enclose h1-h6 heading tag inside. It is a semantic or structural tag
Unlike the h1-h6 tags, It does not make text bold or big.
Technically speaking, It is used to indicate that this is the head of a document

Input Tag and it's types

The input tag has 6 main different types of data that can be entered in it's field. Web browsers and Integrated Dev Environments(IDEs) are modified to understandand react accordingly to the type of an input

Navigation Tag and Use

A nav tag is used to build a navigation bar that has a lot of links inside it in an orderly structure. The most common tags you can find in a navigation tag is the DIV and anchor/linking tag
  • Div is used to hold all the text in a enclosed format and make things ordered
  • A or anchor tag is used to link to the pages contained in your website

Ways of Adding Colors with Css

There are three main ways of adding colors using CSS to an html element

  • Using the normal Everyday Color style: Red, Green, Blue, Brown, Black and so on.
    • Example p{ color: red; }
  • Using The Hexadecimal-Values of Adding Color. They are written in the following format: #rrggbb. Where rr,gg and bb are values between 00 and ff. rr-the amount of red, gg-the amount of green,bb-the amount of blue. 00 is the lowest and ff is the highest . #000000 forms black , and #FFFFFF forms white. This method offers more color ranges and combinations Example p{ color: #696969; This would form grayish color when applied to an html element }
  • The last way to give html elements color is using RGBA syntax. This one is more straight-forward than the hex-values In rgb, r is red, g is green, b is blue and a is alpha. which means the intensity of the specified color. Do u want it to be totally transparent or opaque,, that's what aplha means. it's a number between 0.0 and 1.0. For the rgb values, 0 depicts the lowest intensity and 255 the highest. The format for writing colors with rgba is rgba(r,g,b,a) where rgb can be any values between0 and 255 and a values between 0.0 and 1.0. Examples p{ color: rgba(255,153,63,0.5) }
And those are all the ways you can define html colors using CSS.
